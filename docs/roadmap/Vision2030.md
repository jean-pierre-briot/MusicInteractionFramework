# Vision 2030

By 2030, MusicInteractionFramework could become:

- a mature open-source framework
- a platform for comparing temporal interaction theories
- a reference implementation for musical interaction analysis
- a bridge between computational musicology, HCI, AI and multi-agent systems
- a reproducible research environment

Potential domains:

- human/AI music improvisation
- ensemble interaction
- dance/music interaction
- human-robot interaction
- conversational dynamics
- multimodal performance analysis
