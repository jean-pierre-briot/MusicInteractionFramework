# ADR-009 — Scientific Positioning

## Status

Accepted.

## Context

There are already many theories and models of temporal interaction.

MusicInteractionFramework should not claim to replace these approaches.

## Decision

Position MusicInteractionFramework as a meta-framework.

It provides a common ontology and architecture in which existing and new temporal interaction theories can be implemented, compared and evaluated.

## Consequences

The framework remains theoretically open.

The plugin architecture becomes central.

Scientific documentation becomes part of the project, not an afterthought.
