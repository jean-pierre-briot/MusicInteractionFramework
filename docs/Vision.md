# MusicInteractionFramework Vision

MusicInteractionFramework models, analyses and visualises interactions between temporal agents.

The project originated from the analysis of musical interactions between a human player and the Continuator, but its scope is broader.

Core principles:

- The framework is independent from MIDI, Logic Pro, the Continuator or any specific software.
- Agents produce sequences.
- Sequences contain temporal elements.
- Interaction models generate relations between sequences.
- Metrics are applied later and produce metric results.
- Visualisation is a separate layer.
