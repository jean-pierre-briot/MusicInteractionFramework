# Framework Invariants

1. Every conceptual framework object is an Entity or is associated with one.
2. A Sequence is produced by one Agent.
3. A Relation connects two Sequences.
4. A Relation does not define its own semantic theory.
5. A Session is a container of entities.
6. A RelationGraph is a view over Relations.
7. Metrics do not mutate analysed objects.
8. Metrics produce MetricResult objects.
9. Derived objects should preserve Provenance.
10. The framework must allow multiple theories of temporal interaction to coexist.
